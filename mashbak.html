Mashbak AI SMS Assistant

Mashbak is a personal AI assistant system developed by Mattis Klein. Approved users can interact with the assistant via SMS to ask questions, receive information, and perform limited automated tasks.

Opt-In

Users opt in by sending a keyword or request message such as:

  • @bucherim
  • join@bucherim

Only approved numbers are allowed to interact with the system.

Opt-Out

Users may stop messages at any time by replying:

STOP

Message Frequency

Message frequency varies depending on user interaction. The system only responds to user-initiated requests and does not send marketing or bulk messages.
