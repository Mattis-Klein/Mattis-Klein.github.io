Mashbak AI SMS Assistant

Mashbak is a personal AI assistant system developed by Mattis Klein. Approved users can interact with the assistant via SMS to ask questions, receive information, and perform limited automated tasks. The system only responds to user‑initiated requests and does not send marketing or bulk messages.

Opt‑In

Users opt in by sending the keyword:

join@bucherim

When this keyword is sent, the request is processed and if valid, the user is added to the list of approved users.
By sending a message to the system, users consent to receive responses from the assistant. Access is restricted to approved numbers only. Message and data rates may apply. Users may reply STOP to opt out at any time or HELP for assistance.

Opt‑Out

Users may stop messages at any time by replying:

STOP

By replying STOP, users will no longer receive responses from the assistant.
Reply HELP for assistance.

Message Frequency

Message frequency varies depending on user interaction. The system only responds to messages sent by approved users and does not send marketing, notifications, or bulk messaging campaigns.
