HTML Entities

HTML entities are special codes that represent characters in HTML. They are used to display characters that have a special meaning in HTML, such as <,>, &, ", and '.

Here are some common HTML entities:

  • &amp; - Ampersand (&)
  • &lt; - Less than (<)
  • &gt; - Greater than (>)
  • &quot; - Double quotation mark (")
  • &apos; - Single quotation mark (')
  • &copy; - Copyright symbol (©)
  • &reg; - Registered trademark symbol (®)
  • &nbsp; - Non-breaking space
